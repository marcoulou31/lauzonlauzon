# Photos des propriétés vendues

Déposer ici les photos envoyées par la cliente (format .jpg ou .webp).

Ensuite, dans `src/data/soldProperties.ts`, remplacer les `src` d'exemple
(Unsplash) par le chemin local, par exemple :

    src: "/proprietes-vendues/ma-photo.jpg",

et ajuster `alt` (accessibilité) et `caption` (courte description) de chaque photo.
